Hi, I'm Fritz!

Full Stack Web Developer | Data-Driven Problem Solver

I'm a developer focused on building scalable web applications and data-driven systems. My work bridges frontend development, backend architecture, and analytics to create solutions that are not just functional, but operationally useful.

I've worked with modern web stacks including React and Node.js, and I also build data workflows and dashboards using Python (NumPy, Pandas, Sci-kit), SQL, and visualization tools to support better decision-making.

I'm also actively learning cybersecurity fundamentals to better understand how systems are secured and how they can be improved.
